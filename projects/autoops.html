AIOps · 时间序列预测

AutoOps 云监控预测与异常告警平台

结合 AutoML 与可解释模型，为 SRE 团队提供 GPU 指标预测与实时异常侦测，显著降低误报并提升故障响应效率。

关键数据

  • 告警噪音下降 35%
  • 预测准确度 (MAPE) 10.6%
  • 节约 GPU 成本 12%
  • 上线后 3 个月服务 180+ 告警策略

"AIOps 的价值，在于把预测与解释交到一线运维手中。"

结果与贡献

聚焦异常检测稳定性、可解释能力与业务落地。

异常检测

  • 引入 Prophet + LSTM 混合模型，自适应节假日与活动波动
  • 自监督对比学习识别冷启动与漂移场景
  • 异常分级策略减少 42% 非关键告警

可解释性

  • Shapley 模块输出 Top-K 影响因子
  • Grafana 插件提供 root cause drill-down
  • 建立“承诺-验证”机制记录运维反馈

工程实践

  • 使用 Kubeflow Pipelines 管理训练与部署
  • Prometheus 指标接入 PagerDuty 自动化响应
  • CI/CD 接入 GitOps，平均迭代周期缩短 30%

预测 + 告警配置

技术实现

流程覆盖数据采集、特征工程、训练调度与在线推理。

Pipeline

AutoML 训练流程

利用自研 AutoML 框架在 Kubeflow 上执行特征选择与模型训练，支持快速扩展到新指标。

  • TSFresh + 自建指标库生成 120+ 特征
  • 模型候选包括 Prophet、TFT、LSTM、XGBoost
  • 自动模型组合与 stacking 提升 MAPE 7%
Serving

在线推理与监控

采用 FastAPI + Celery 构建实时推理服务，Prometheus 采集指标并进行异常告警。

  • Redis 作消息队列，支持批量推理
  • OpenTelemetry 追踪跨服务延迟
  • 与 PagerDuty 集成，异常即刻通知值班人员
